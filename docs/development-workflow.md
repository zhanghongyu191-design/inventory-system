# Development Workflow

## 1. Pull latest code

```bash
git pull origin main
```

## 2. Create Issue

Create GitHub Issue before development.

## 3. Create Feature Branch

```bash
git checkout -b feature/issue-xx
```

## 4. Development

Implement feature.

## 5. Commit

```bash
git add .
git commit -m "feat: ..."
```

## 6. Push

```bash
git push origin feature/issue-xx
```

## 7. Pull Request

Create Pull Request.

## 8. Merge

Merge into main after review.
